# SQL Data Analysis Project

This repository contains SQL scripts developed as part of my practice and learning in database management, querying, and data analysis using MySQL Workbench.

 Project Overview
The project is based on a sample retail database (`sql_store`) that includes real-world entities such as:

- Customers  
- Orders  
- Order Items  
- Products  
- Shippers  

Using this data, I performed various SQL operations, including:

- Joins  
- Subqueries  
- Grouping and Aggregation  
- Window Functions  
- Date-based analysis  
- Data insights and reporting
